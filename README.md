# laser-surface-measure

##一个基于opencv和numpy制作的基于海康威视sdk和相机的激光平面标定程序

##目前包含拍摄及单目标定、激光平面标定功能

1、目前支持海康威视全系列相机
2、参数保存采用json文件
3、全标定部分独立运作，其他相机可拍摄后放到对应文件夹运行标定功能
4、给出简易测试软件
5、后续待更新一个单纯的更易用的万用标定程序
6、预期加入多目标定功能，及其他激光分割标定
